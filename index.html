ABOUT ME

The name is Hammed and I facilitate learning in Maths and ICT at Alhijrah College in Ibadan. My love for IT started when I had to find a solution to a timetabling issue in my former work place. Since then I have developed a strong affection for anything that's written in code.

I started my code journey with a course-Programming for Everybody by Charles Severance on Coursera. I love playing around UI and UX learning HTML5, CSS3, JavaScript and Python. I care so much about kids learning to code at an early age.

I use my off-work time honing my coding skills, working on my e-learning project website PractixAM which is being developed to cater for the growing demands of free online cbt practices. Please feel free to check it out though still a work in progress .
